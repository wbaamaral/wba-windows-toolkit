| Prefixo    | Tradução              | Quando usar                                               |
| ---------- | --------------------- | --------------------------------------------------------- |
| `feat`     | Nova funcionalidade   | Adição de um novo script ou recurso                       |
| `fix`      | Correção              | Correção de bug ou comportamento incorreto                |
| `docs`     | Documentação          | README, manuais, exemplos, comentários                    |
| `style`    | Formatação            | Ajustes de identação, espaços, layout, sem alterar lógica |
| `refactor` | Refatoração           | Reorganização do código sem mudar funcionalidade          |
| `perf`     | Otimização            | Melhorias de desempenho                                   |
| `test`     | Testes                | Inclusão ou ajuste de testes                              |
| `build`    | Processo de build     | Alterações de empacotamento ou geração                    |
| `ci`       | Integração contínua   | GitHub Actions, Gitea Actions, pipelines                  |
| `chore`    | Tarefa administrativa | Mudanças de suporte, sem impacto funcional                |
| `revert`   | Reversão              | Desfaz um commit anterior                                 |
| `security` | Segurança             | Correções ou melhorias de segurança                       |
| `deps`     | Dependências          | Atualização de dependências ou módulos                    |
| `release`  | Publicação            | Preparação de versão ou release                           |


Exemplos para o projeto:

| Commit                                                                          | Significado           |
| ------------------------------------------------------------------------------- | --------------------- |
| `feat(updates): adiciona rotina de atualização via Windows Update e Chocolatey` | Novo recurso          |
| `feat(maintenance): adiciona script de limpeza segura do Windows`               | Novo recurso          |
| `docs(readme): adiciona documentação inicial do projeto`                        | Documentação          |
| `docs: atualiza instruções de instalação`                                       | Documentação          |
| `fix(updates): corrige detecção do Chocolatey`                                  | Correção              |
| `fix(maintenance): corrige limpeza de arquivos temporários`                     | Correção              |
| `refactor(common): padroniza funções de logging`                                | Refatoração           |
| `perf(maintenance): otimiza remoção de arquivos temporários`                    | Otimização            |
| `chore: ativa suporte a Git LFS`                                                | Tarefa administrativa |
| `chore: adiciona estrutura inicial de diretórios`                               | Tarefa administrativa |
| `chore: atualiza licença MIT`                                                   | Tarefa administrativa |
| `security(updates): valida privilégios administrativos antes da execução`       | Segurança             |
| `deps: atualiza dependências do ambiente de desenvolvimento`                    | Dependências          |
| `release: versão 1.0.0`                                                         | Publicação            |

feat(maintenance): adiciona script de limpeza segura do Windows
feat(updates): adiciona atualização via Chocolatey
fix(networking): corrige validação de gateway
docs(readme): adiciona documentação inicial
refactor(common): padroniza tratamento de erros
chore: ativa suporte a Git LFS
security(active-directory): valida permissões antes da execução
