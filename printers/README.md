# printers

Scripts de gerenciamento de impressoras em ambientes Windows. Instalação, remoção, diagnóstico de filas de impressão, gerenciamento de drivers e configuração de impressoras compartilhadas via servidor de impressão.

> **Em desenvolvimento.** Scripts planejados para este diretório:
>
> - Instalação de impressoras por IP ou nome de compartilhamento
> - Limpeza de fila de impressão travada (spooler)
> - Listagem e exportação de impressoras instaladas
> - Mapeamento de impressoras por GPO/logon script
> - Diagnóstico de drivers de impressora
