# Portal do Operador

Entrada principal do `Xtudo` para o operador.

## Caminho de uso

1. Abra o PowerShell na raiz do repositório.
2. Execute `.\xtudo.ps1`.
3. Use `1` a `5` para atalhos rápidos.
4. Use uma palavra-chave para buscar uma ação.
5. Use `0`, `q` ou `sair` para cancelar.

## Ações principais

- `Limpar Windows`
- `Diagnosticar disco 100%`
- `Diagnosticar memória`
- `Diagnosticar gráfico`
- `Diagnosticar AD cliente`
- `Preparar imagem`

## Quando o operador estiver em dúvida

- Pressione `Enter` para listar tudo.
- Prefira o launcher em vez de chamar scripts legados direto.
- Use a documentação técnica apenas quando precisar localizar implementação.
