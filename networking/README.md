# networking

Scripts de configuração e diagnóstico de rede. Abrangem configuração de adaptadores, proxy, DNS, firewall, compartilhamentos e troubleshooting de conectividade em ambientes corporativos com Active Directory.

> **Em desenvolvimento.** Scripts planejados para este diretório:
>
> - Diagnóstico completo de adaptadores de rede
> - Configuração de proxy por GPO ou script
> - Renovação e liberação de DHCP / flush DNS
> - Teste de portas e firewall corporativo
> - Mapeamento de drives de rede
