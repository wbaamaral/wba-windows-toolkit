# optimization

Scripts de otimização de desempenho do Windows. Ajustes de configurações visuais, planos de energia, prioridades de processo, serviços desnecessários e outras otimizações conservadoras que aumentam a responsividade do sistema sem comprometer sua estabilidade.

> **Em desenvolvimento.** Scripts planejados para este diretório:
>
> - Configuração de plano de energia para alto desempenho
> - Desativação de efeitos visuais desnecessários
> - Revisão de programas na inicialização
> - Ajuste de configurações de memória virtual
